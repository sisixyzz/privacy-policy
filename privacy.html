Kebijakan Privasi

Aplikasi ini dibuat oleh pemilik untuk penggunaan pribadi, dan tidak mengumpulkan atau membagikan data pengguna ke pihak ketiga.

Aplikasi ini hanya digunakan untuk memposting konten ke halaman Facebook milik pribadi secara otomatis melalui API resmi Facebook.

Tidak ada data pengguna yang dikumpulkan, disimpan, atau digunakan di luar fungsi utama aplikasi.

Jika Anda memiliki pertanyaan terkait privasi, silakan hubungi pemilik aplikasi secara langsung.
